# PDB Structure Parser

Parse and analyze protein 3D structures from PDB files.

## Overview

This project implements a PDB (Protein Data Bank) file parser for structural bioinformatics analysis.

## Tech Stack

- Python 3.x
- Standard library

## Features

- PDB file reading
- Custom column extraction system
- ATOM line parsing

## Setup

```bash
pip install -r requirements.txt
```

## Usage

```bash
python pdb_parser.py <path_to_pdb_file>
```
